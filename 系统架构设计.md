## 一、项目概述

### 1.1 项目简介

血压守护是一款专注于高血压患者及其家庭成员的智能健康管理平台，采用前后端分离架构，支持多终端访问（Web浏览器、PWA应用、微信小程序）。

### 1.2 技术特点

- **轻量级架构**：Flask后端 + 原生JavaScript前端，无需复杂构建工具
- **跨平台兼容**：PWA架构，一次开发，多端运行
- **离线可用**：Service Worker + LocalStorage实现离线功能
- **数据安全**：密码加密、HTTPS、CSRF/XSS防护
- **高可扩展**：模块化设计，易于扩展新功能

### 1.3 核心模块

```
血压守护
├── 前端模块
│   ├── 用户界面（HTML/CSS/JS）
│   ├── 本地存储（LocalStorage）
│   ├── 离线支持（Service Worker）
│   └── 数据可视化（Canvas/ECharts）
│
├── 后端模块
│   ├── API接口（Flask Routes）
│   ├── 业务逻辑（Services）
│   ├── 数据模型（Models）
│   └── 安全模块（Security）
│
└── 基础设施
    ├── 数据库（MySQL/SQLite）
    ├── 缓存（Redis）
    ├── 日志（Python Logging）
    └── 部署（Nginx + Gunicorn）
```

### 3.2 命名规范

#### 后端命名规范
- **文件名**：小写 + 下划线（snake_case），如：`auth.py`、`user_service.py`
- **类名**：大驼峰（PascalCase），如：`User`、`MedicationService`
- **函数名**：小写 + 下划线（snake_case），如：`get_user_by_id()`
- **常量**：大写 + 下划线，如：`MAX_RETRY_COUNT`
- **变量**：小写 + 下划线，如：`user_name`

#### 前端命名规范
- **文件名**：小写 + 下划线，如：`app_pro.html`
- **变量名**：小驼峰（camelCase），如：`userName`
- **函数名**：小驼峰，如：`saveData()`
- **常量**：大写 + 下划线，如：`MAX_RECORDS`
- **CSS类名**：小写 + 连字符，如：`.user-info`

### 5.2 表结构详细说明

#### 用户表（users）

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(80) NOT NULL UNIQUE,
    password_hash VARCHAR(256) NOT NULL,
    phone VARCHAR(20) UNIQUE,
    email VARCHAR(120) UNIQUE,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    is_active BOOLEAN DEFAULT TRUE,
    INDEX idx_username (username),
    INDEX idx_phone (phone)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 家庭成员表（family_members）

```sql
CREATE TABLE family_members (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    age INT,
    gender VARCHAR(10),
    relation VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    INDEX idx_user_id (user_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 药物表（medications）

```sql
CREATE TABLE medications (
    id INT AUTO_INCREMENT PRIMARY KEY,
    member_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    dosage VARCHAR(100),
    frequency VARCHAR(50),
    times TEXT COMMENT 'JSON格式存储服药时间',
    quantity INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (member_id) REFERENCES family_members(id) ON DELETE CASCADE,
    INDEX idx_member_med (member_id, name)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 血压记录表（blood_pressure_records）

```sql
CREATE TABLE blood_pressure_records (
    id INT AUTO_INCREMENT PRIMARY KEY,
    member_id INT NOT NULL,
    systolic INT NOT NULL COMMENT '收缩压',
    diastolic INT NOT NULL COMMENT '舒张压',
    heart_rate INT COMMENT '心率',
    note TEXT COMMENT '备注',
    record_time DATETIME NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (member_id) REFERENCES family_members(id) ON DELETE CASCADE,
    INDEX idx_member_time (member_id, record_time),
    INDEX idx_record_time (record_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 预警表（alerts）

```sql
CREATE TABLE alerts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    member_id INT NOT NULL,
    log_id INT,
    med_name VARCHAR(100),
    diff INT COMMENT '偏差值',
    level VARCHAR(20) COMMENT 'normal/warn/urgent/critical',
    status VARCHAR(20) DEFAULT 'active',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    resolved_at DATETIME,
    FOREIGN KEY (member_id) REFERENCES family_members(id) ON DELETE CASCADE,
    INDEX idx_member_status (member_id, status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 协作日志表（collaboration_logs）

```sql
CREATE TABLE collaboration_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    member_id INT NOT NULL,
    action VARCHAR(100) NOT NULL,
    detail TEXT,
    operator VARCHAR(100),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (member_id) REFERENCES family_members(id) ON DELETE CASCADE,
    INDEX idx_member_time (member_id, created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 5.3 索引优化策略

| 表名 | 索引类型 | 字段 | 用途 |
|------|----------|------|------|
| users | UNIQUE | username | 快速登录验证 |
| users | UNIQUE | phone | 手机号登录 |
| family_members | INDEX | user_id | 查询用户家庭成员 |
| medications | COMPOSITE | (member_id, name) | 查询成员药物 |

| alerts | COMPOSITE | (member_id, status) | 查询成员预警 |
| collaboration_logs | COMPOSITE | (member_id, created_at) | 查询协作日志 |

## 九、安全设计

### 9.1 密码安全

```python
# app/security.py
import hashlib
import secrets

class PasswordHasher:
    """密码加密和验证"""
    
    @staticmethod
    def hash_password(password):
        """加密密码"""
        salt = secrets.token_hex(16)
        hashed = hashlib.pbkdf2_hmac(
            'sha256',
            password.encode('utf-8'),
            salt.encode('utf-8'),
            100000  # 迭代次数
        )
        return f"{salt}${hashed.hex()}"
    
    @staticmethod
    def verify_password(password, hashed):
        """验证密码"""
        try:
            salt, hash_value = hashed.split('$')
            new_hash = hashlib.pbkdf2_hmac(
                'sha256',
                password.encode('utf-8'),
                salt.encode('utf-8'),
                100000
            )
            return hmac.compare_digest(new_hash.hex(), hash_value)
        except Exception:
            return False
```

### 9.2 XSS防护

```python
class XSSProtector:
    """XSS攻击防护"""
    
    DANGEROUS_PATTERNS = [
        r'<script[^>]*>.*?</script>',
        r'javascript:',
        r'on\w+\s*=',
        r'<iframe[^>]*>.*?</iframe>',
    ]
    
    @classmethod
    def sanitize(cls, text):
        """清理XSS攻击代码"""
        if not text:
            return text
        
        for pattern in cls.DANGEROUS_PATTERNS:
            text = re.sub(pattern, '', text, flags=re.IGNORECASE | re.DOTALL)
        
        # HTML实体编码
        text = text.replace('&', '&amp;')
        text = text.replace('<', '&lt;')
        text = text.replace('>', '&gt;')
        text = text.replace('"', '&quot;')
        
        return text
```

### 9.4 速率限制

```python
class RateLimiter:
    """API速率限制"""
    
    def __init__(self):
        self.requests = {}
    
    def is_allowed(self, key, max_requests=100, window=3600):
        """检查是否允许请求"""
        now = datetime.now()
        
        if key not in self.requests:
            self.requests[key] = []
        
        # 清理过期记录
        self.requests[key] = [
            t for t in self.requests[key]
            if (now - t).total_seconds() < window
        ]
        
        if len(self.requests[key]) >= max_requests:
            return False
        
        self.requests[key].append(now)
        return True

def rate_limit(max_requests=100, window=3600):
    """速率限制装饰器"""
    def decorator(f):
        @wraps(f)
        def decorated_function(*args, **kwargs):
            client_ip = request.remote_addr
            if not rate_limiter.is_allowed(client_ip, max_requests, window):
                return jsonify({
                    'success': False,
                    'message': '请求过于频繁，请稍后再试'
                }), 429
            return f(*args, **kwargs)
        return decorated_function
    return decorator
```
