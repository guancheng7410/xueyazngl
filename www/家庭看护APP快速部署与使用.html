一、应用简介

血压守护是一款专为高血压患者及家属设计的智能健康管理应用，提供以下核心功能：

  • 💊 药物管理与智能服药提醒
  • 🩺 血压记录与趋势图表分析
  • 👨‍👩‍👧 家庭成员协同管理
  • 🔔 多级预警与推送通知
  • 📊 健康评估与数据统计
  • 💾 离线使用，数据安全存储在本机

二、系统要求

项目要求说明
Android手机Android 6.0+推荐使用Chrome浏览器
iPhoneiOS 11.0+推荐使用Safari浏览器
存储空间至少50MBPWA方式占用空间极小
网络安装时需要联网安装后可离线使用

三、安装方式一：PWA安装（⭐ 推荐）

✅ 为什么推荐？
• 无需下载APK文件
• 安装过程仅需3步
• 自动更新，始终保持最新版本
• 支持离线使用
• 不占用手机存储空间

📱 Android手机安装步骤

1

在电脑上启动服务器

双击运行项目目录下的 安装到手机.bat 文件

选择选项 1，启动本地服务器

服务器地址：http://您的电脑IP:8082
例如：http://192.168.1.100:8082
💡 提示：查看电脑IP的方法：
1. 按 Win+R 打开运行窗口
2. 输入 cmd 回车
3. 输入 ipconfig 回车
4. 找到"IPv4 地址"，通常是 192.168.x.x
2

在手机浏览器中打开网址

确保手机和电脑连接在同一个WiFi网络

在手机 Chrome浏览器 中输入服务器地址

http://192.168.1.100:8082
🌐

手机浏览器显示血压守护应用界面

3

添加到主屏幕

点击Chrome浏览器右上角的 三个点菜单

在菜单中找到并点击 "添加到主屏幕"

💡 提示：如果没有看到该选项，请确保使用的是Chrome浏览器，并确认网址正确
📲

浏览器菜单显示"添加到主屏幕"选项

4

完成安装

点击 "添加" 确认

返回手机主屏幕，即可看到 "血压守护" 图标

🎉 安装完成！
点击图标即可打开应用，像原生APP一样使用

🍎 iPhone安装步骤

1

在Safari浏览器中打开网址

确保手机和电脑在同一个WiFi网络

在 Safari浏览器 中输入服务器地址

http://192.168.1.100:8082
⚠️ 注意：必须使用Safari浏览器，其他浏览器不支持添加到主屏幕
2

点击分享按钮

Safari浏览器底部工具栏的分享按钮

3

添加到主屏幕

在分享菜单中向下滚动，找到并点击 "添加到主屏幕"

修改名称（可选），然后点击 "添加"

4

完成安装

返回手机主屏幕，即可看到 "血压守护" 图标

🎉 安装完成！
点击图标即可全屏打开应用

四、安装方式二：HBuilderX云打包

🎯 适用场景：需要APK文件，但没有Android开发环境的用户
1

下载HBuilderX

访问官网下载： https://www.dcloud.io/hbuilderx.html

选择 "标准版" 下载（免费）

下载后解压即可使用，无需安装

2

导入项目

打开HBuilderX

点击菜单：文件 -> 导入 -> 从本地目录导入

选择目录： E:\开发需要\blood-pressure-guardian\www

点击"选择"完成导入

3

配置应用信息

在左侧项目树中，双击打开 manifest.json 文件

填写以下信息：

配置项填写内容
应用名称血压守护
应用描述智能服药提醒与血压记录
版本号2.0.0
版本名称2.0.0
4

云端打包

右键点击项目名称

选择：发行 -> 原生App-云打包

在打包界面配置：

  • 平台：选择 Android
  • 包名： com.bloodpressureguardian.app
  • 证书：使用公共测试证书（首次使用）

点击 "打包" 按钮

⏱️ 等待时间：云端打包通常需要 3-5 分钟
5

下载并安装APK

打包完成后，会自动下载APK文件

将APK文件复制到： D:\APP\血压守护.apk

将APK发送到手机，点击安装

🎉 打包完成！
APK文件已保存到 D:\APP 目录

五、安装方式三：Android Studio打包

⚠️ 前置要求：
• 已安装 Node.js 14+
• 已安装 Java JDK 11+
• 已安装 Android Studio
• 已配置 Android SDK
1

安装依赖

打开命令行（Win+R 输入 cmd）

cd E:\开发需要\blood-pressure-guardian
npm install
2

添加Android平台

npx cap add android

等待下载完成，会在项目目录下生成 android 文件夹

3

同步项目

npx cap sync android
4

打开Android Studio

npx cap open android

Android Studio会自动打开项目

⏱️ 首次打开：Gradle同步可能需要 5-10 分钟，请耐心等待
5

构建APK

在Android Studio菜单中点击：

Build -> Build Bundle(s) / APK(s) -> Build APK(s)

等待构建完成，底部会显示提示

6

复制APK到D:\APP

APK位置： android\app\build\outputs\apk\debug\app-debug.apk

复制到： D:\APP\血压守护.apk

🎉 构建成功！
APK文件已保存到 D:\APP 目录

六、在手机上安装APK

如果您已经获得了APK文件，请按以下步骤安装：

1

将APK发送到手机

选择以下任一方式：

  • 📧 通过微信/QQ发送给手机
  • 📱 通过数据线复制到手机
  • ☁️ 上传到云盘，在手机上下载
  • 📶 通过局域网共享
2

允许安装未知来源应用

首次安装时，手机会提示"禁止安装未知来源应用"

点击 "设置"，开启 "允许从此来源安装"

💡 提示：此提示是Android系统的安全机制，血压守护是安全的应用，可以放心安装
3

安装APK

在手机文件管理器中找到APK文件

点击APK文件，点击 "安装"

等待安装完成

4

打开应用

安装完成后，点击 "打开"

或在手机桌面上找到 "血压守护" 图标

🎉 安装成功！
应用已安装到您的手机，可以开始使用了
⚠️ 注意事项：
• 安装时如提示风险警告，请点击"继续安装"
• 此应用为单机版，不会上传任何数据到网络
• 所有数据均存储在您的手机本地

打开应用

九、数据管理与备份

💾 数据导出

1

导出数据

点击底部导航栏的 "我的"

点击 "📊 导出数据"

系统会自动下载JSON格式的备份文件

📋 建议备份频率

使用频率建议备份周期
每天使用每周备份一次
偶尔使用每月备份一次
重要数据每次更新后备份
⚠️ 重要提醒：
• 数据存储在浏览器本地存储中
• 清除浏览器数据会导致数据丢失
• 建议定期导出备份
• 不要点击"清除所有数据"，除非确定要删除

🗑️ 清除数据

如需清除所有数据：

  1. 先点击"📊 导出数据"备份数据
  2. 点击"🗑️ 清除所有数据"
  3. 确认清除操作
⚠️ 警告：清除数据后无法恢复，请务必先备份！

十、常见问题解答

Q1：应用可以离线使用吗？
A：可以。PWA安装后支持离线使用，所有功能无需联网即可正常使用。数据存储在本地，不会上传到网络。
Q2：数据会丢失吗？
A：只要不清除浏览器数据，数据不会丢失。建议定期使用"导出数据"功能进行备份。
Q3：可以在多个设备上使用吗？
A：当前为单机版，数据存储在设备本地。如需在多设备同步，需要手动导出/导入数据文件。
Q4：为什么PWA方式推荐Chrome/Safari浏览器？
A：Chrome和Safari对PWA支持最完善，可以添加到主屏幕并全屏显示，体验最接近原生APP。
Q5：安装时提示"未知来源应用"安全吗？
A：安全。这是Android系统的安全机制，所有非应用商店下载的应用都会提示。血压守护是开源项目，不含任何恶意代码。
Q6：应用占用多少存储空间？
A：PWA方式仅占用约5-10MB空间。APK安装后约占用15-20MB空间。数据存储在本地，根据使用量会逐渐增加。
Q7：如何更新应用到最新版本？
A：PWA方式会自动更新，打开应用即可获取最新版本。APK方式需要重新打包并安装新版本。
Q8：支持哪些手机型号？
A：支持Android 6.0+和iOS 11.0+的所有手机型号。包括华为、小米、OPPO、vivo、三星、iPhone等主流品牌。
Q9：服药提醒准时吗？
A：PWA方式依赖浏览器通知，需要保持浏览器在后台运行。APK方式可以使用系统级通知，提醒更准时。
Q10：如何联系技术支持？
A：如有问题，请查看项目文档，或通过项目仓库提交Issue。我们会尽快回复您的问题。
