📱 血压守护

智能服药提醒与血压记录系统 - 安装指南

✨ 推荐方式：PWA 安装（最简单）

无需APK文件，直接在手机浏览器中安装，像原生APP一样使用


1
双击运行 "安装到手机.bat"，启动本地服务器
2
在手机浏览器中访问：
http://您的电脑IP:8082
（IP地址可通过命令行运行 ipconfig 查看）
3
点击浏览器菜单中的 "添加到主屏幕"
4
完成！现在可以在手机主屏幕找到"血压守护"图标
✅ 优点：无需编译环境 | 自动更新 | 支持离线使用 | 不占用手机存储空间

🔧 方式二：使用 HBuilderX 云打包 APK

适合没有Android开发环境但需要APK文件的用户


1
下载 HBuilderX：https://www.dcloud.io/hbuilderx.html
2
打开 HBuilderX，导入 www 文件夹
3
右键项目 -> 发行 -> 原生App-云打包
4
选择 Android，配置应用信息，点击"打包"
5
等待云端打包完成（约3-5分钟），下载APK
6
将APK复制到 D:\APP\血压守护.apk

💻 方式三：使用 Android Studio 本地打包

适合有Android开发经验的用户


环境要求：
• Node.js 14+
• Java JDK 11+
• Android Studio

在命令行中依次执行：

cd E:\开发需要\blood-pressure-guardian
npm install
npx cap add android
npx cap sync android
npx cap open android

在 Android Studio 中：

1
等待 Gradle 同步完成
2
Build -> Build Bundle/APK -> Build APK
3
APK位置：android\app\build\outputs\apk\debug\app-debug.apk
4
复制到 D:\APP\血压守护.apk
